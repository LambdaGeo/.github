# LambdaGeo 🔷

> Semantic technologies and geospatial intelligence for environmental modeling

[![License: MIT](https://img.shields.io/badge/License-MIT-blue.svg)](LICENSE)
[![Research](https://img.shields.io/badge/Research-Geoinformatics-green)](https://lambdageo.github.io)

## 🎯 Mission
Developing open-source tools that bridge Linked Data, spatial analysis, and environmental science — with focus on coastal dynamics, mangrove monitoring, and sustainable development.

## 🔬 Key Projects
| Repository | Description | Tech Stack |
|-----------|-------------|-----------|
| [qgissparql](link) | SPARQL integration for QGIS | Python, RDF, Qt |
| [brmangue](link) | Brazilian mangrove semantic dataset | GeoJSON, SHACL, CKAN |
| [coastal-dynamics](link) | Simulation framework for erosion modeling | Python, Xarray, Dask |
